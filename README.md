# مدیریت طلافروشی (PWA)

## این چیه؟
یه اپ وب نصب‌شونی (PWA) کامل برای اداره‌ی روزانه‌ی یه طلافروشی: محاسبه‌ی قیمت، صدور فاکتور فروش، ثبت خرید طلای کهنه، مدیریت موجودی کالا، دفترچه‌ی مشتریان، تاریخچه‌ی فاکتورها و گزارش فروش/سود.

## قابلیت‌ها
- **محاسبه**: فرمول کامل با اجرت/سود/مالیات + محاسبه از روی مظنه‌ی جهانی (اونس و نرخ دلار)
- **فروش**: انتخاب مشتری یا افزودن مشتری جدید، افزودن قلم از موجودی (که خودکار کم می‌شه) یا دستی، صدور فاکتور با شماره‌ی خودکار و جای مهر
- **خرید کهنه**: محاسبه‌ی مبلغ پرداختی بر اساس وزن، عیار و درصد کسر، امکان افزودن خودکار به موجودی
- **موجودی**: ثبت کالا با کد، عیار، وزن و تعداد؛ جست‌وجو
- **مشتریان**: ثبت نام و تماس، نمایش تعداد و جمع خریدهای هر مشتری
- **فاکتورها**: تاریخچه‌ی همه‌ی فاکتورهای فروش و خرید با امکان چاپ مجدد
- **گزارش**: جمع فروش، سود و خرید کهنه برای امروز / این ماه / کل
- **تنظیمات**: نام و آدرس فروشگاه (روی فاکتور چاپ می‌شه)، و اتصال اختیاری به قیمت لحظه‌ای اونس

## نکته‌ی مهم درباره‌ی قیمت لحظه‌ای
تب تنظیمات یه گزینه‌ی «دریافت قیمت لحظه‌ای اونس» داره که به سرویس goldapi.io وصل می‌شه (نیاز به یه کلید رایگان از سایتشون داره). اما:
- نرخ دلار به تومان (بازار آزاد ایران) سرویس رایگان معتبری نداره، پس همیشه باید دستی وارد بشه
- بعضی مرورگرها ممکنه این اتصال مستقیم رو به‌خاطر CORS مسدود کنن — اگه کار نکرد، همون روش دستی (وارد کردن قیمت اونس در تب محاسبه) رو استفاده کن

## چطور نصبش کنم روی گوشی؟

برای اینکه دکمه‌ی "نصب" روی گوشی ظاهر بشه، این فایل‌ها باید روی یه هاست HTTPS باشن (فایل local باز کردن کافی نیست). ساده‌ترین راه‌ها:

### روش ۱: GitHub Pages (رایگان)
1. یه ریپازیتوری جدید توی گیت‌هاب بساز
2. همه‌ی فایل‌های این پوشه (index.html, manifest.json, service-worker.js, icons/) رو آپلود کن
3. برو توی Settings > Pages و فعالش کن (Branch: main)
4. لینکی که می‌ده رو توی مرورگر گوشیت (Chrome) باز کن
5. توی منوی مرورگر گزینه‌ی «Add to Home screen» یا «Install app» رو بزن

### روش ۲: Netlify Drop (سریع‌ترین، بدون حساب گیت‌هاب)
1. برو به netlify.com/drop
2. کل پوشه رو بکش و بنداز داخل صفحه
3. یه لینک آماده می‌ده، همون رو روی گوشی باز کن و نصب کن

## استفاده از فاکتور
توی تب «فروش»، مشتری و اقلام رو وارد کن، دکمه‌ی «ثبت و چاپ / PDF» رو بزن — هم فاکتور ذخیره می‌شه و هم گوشی گزینه‌ی «Save as PDF» رو توی دیالوگ چاپ نشون می‌ده.

## داده‌ها
همه‌چیز (موجودی، مشتریان، فاکتورها، تاریخچه) روی خود گوشی (localStorage مرورگر) ذخیره می‌شه، نه سرور. یعنی خصوصیه ولی اگه cache مرورگر رو پاک کنی از بین می‌ره.

## اگه قبلاً نصبش کرده بودی و آپدیت نمی‌شه
این اپ آفلاین هم کار می‌کنه، برای همین گوشی فایل‌ها رو کش (ذخیره) می‌کنه. بعد از هر آپدیت روی گیت‌هاب:
1. اول مطمئن شو فایل‌های جدید واقعاً جایگزین فایل‌های قبلی روی گیت‌هاب شدن (نه کنارشون)
2. اپ نصب‌شده رو از روی گوشی حذف کن (Uninstall)
3. توی Chrome، تنظیمات سایت (Site settings) برای آدرس گیت‌هاب‌پیجزت رو باز کن و «Clear & reset» / پاک کردن داده‌های سایت رو بزن
4. صفحه رو دوباره باز کن و از اول نصب کن
