# ماشین‌حساب و فاکتور طلا (PWA)

## این چیه؟
یه اپ وب نصب‌شونی (PWA) هست. یعنی بدون رفتن به Google Play، مستقیم از مرورگر روی گوشی اندرویدت نصب می‌شه و مثل یه اپ واقعی (با آیکون، تمام‌صفحه، حتی آفلاین) کار می‌کنه.

## چطور نصبش کنم روی گوشی؟

برای اینکه دکمه‌ی "نصب" روی گوشی ظاهر بشه، این فایل‌ها باید روی یه هاست HTTPS باشن (فایل local باز کردن کافی نیست). ساده‌ترین راه‌ها:

### روش ۱: GitHub Pages (رایگان)
1. یه ریپازیتوری جدید توی گیت‌هاب بساز
2. همه‌ی فایل‌های این پوشه (index.html, manifest.json, service-worker.js, icons/) رو آپلود کن
3. برو توی Settings > Pages و فعالش کن (Branch: main)
4. لینکی که می‌ده رو توی مرورگر گوشیت (Chrome) باز کن
5. توی منوی مرورگر گزینه‌ی «Add to Home screen» یا «Install app» رو بزن

### روش ۲: Netlify Drop (سریع‌ترین، بدون حساب گیت‌هاب)
1. برو به netlify.com/drop
2. کل پوشه رو بکش و بنداز داخل صفحه
3. یه لینک آماده می‌ده، همون رو روی گوشی باز کن و نصب کن

## استفاده از فاکتور
توی تب «فاکتور»، اطلاعات مشتری و کالا رو وارد کن، دکمه‌ی «چاپ / ذخیره PDF» رو بزن — گوشی اندروید خودش گزینه‌ی «Save as PDF» رو توی دیالوگ چاپ نشون می‌ده.

## اگه قبلاً نصبش کرده بودی و آپدیت نمی‌شه
این اپ آفلاین هم کار می‌کنه، برای همین گوشی فایل‌ها رو کش (ذخیره) می‌کنه. بعد از هر آپدیت روی گیت‌هاب:
1. اول مطمئن شو فایل‌های جدید واقعاً جایگزین فایل‌های قبلی روی گیت‌هاب شدن (نه کنارشون)
2. اپ نصب‌شده رو از روی گوشی حذف کن (Uninstall)
3. توی Chrome، تنظیمات سایت (Site settings) برای آدرس گیت‌هاب‌پیجزت رو باز کن و «Clear & reset» / پاک کردن داده‌های سایت رو بزن
4. صفحه رو دوباره باز کن و از اول نصب کن

## تاریخچه
تاریخچه‌ی محاسبات روی خود گوشی (localStorage مرورگر) ذخیره می‌شه، نه سرور. یعنی خصوصیه ولی اگه cache مرورگر رو پاک کنی از بین می‌ره.
